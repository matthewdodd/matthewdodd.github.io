28 April 2017 - Looking at the Old Bailey API

  • The Old Bailey API seems to be the starting point for this project.
  • The first issue, however, is that when the initial query is run it appears to show that there are some 40,521 cases where the sentence was "transportation."
  • The API will only allow generation of up to 1,000 case numbers which leaves over 39,000 not being pulled.
  • The first issue to be solved, therefore, is how to get all 40,521 case numbers for later data mining?
    • Once we have the case numbers it will be simple to Wget all of the cases because we can generate them all based on the API.
  • All of the cases are formatted in XML / JSON which leads me to believe that the data should be straightforward enough to deal with.
  • What exactly has been tagged in the JSON, though? Is it enough for me to perform my research? Will I need to do additional tagging?
  • London Lives is another matter entirely, however. They have no built in API and so I'll have to look and see if I can cobble something together that does the searching for me.
