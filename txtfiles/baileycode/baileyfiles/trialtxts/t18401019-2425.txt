



2425. 




CAROLINE MEARS
 was indicted

for stealing, on the 
25th of August, 4 shawls, value 1l. 5s.; 2 gowns, value 5s.; 5 handkerchiefs, value 5s.; 2 boas, value 1l.; 1 veil, value 1s. 6d.; 4 pairs of shoes, value 8s.; 1 pair of boots, value 1s. 6d.; 1 petticoat, value 1s. 6d.; 1 gown skirt, value 4s.; 3 frocks, value 8s.; 1 pair of bracelets, value 5s.; 1 pair of stockings, value 1s.; 1 collar, value 1s.; 1 reticule, value 1s.; and 2 pincushions, value 2s.; the goods of 



John Edward Fisher, her master; to which she pleaded




GUILTY
. Aged 18.—


Transported for Seven Years—



Penitentiary.
