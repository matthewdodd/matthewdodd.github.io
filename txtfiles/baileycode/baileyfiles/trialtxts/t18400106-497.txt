



497. 




WILLIAM HELLIWELL
was indicted

 for stealing, on the 
20th of December, 4 hammers, value 5s.; 3lbs, weight of tallow, value 1s. 6d.; 1 quart of oil, value 8d.; 1 axe, value 3s.; 5lbs. weight of candles, value 3s.; 20 boxes of matches, value 1s.; 7lbs. weight of emery, value 1s.; 141bs. weight of hone-hair, value 9s.; 8lbs. weight of flax, value 6s.; 1 pair of pincers, value 1s. 6d.; 2 flies, value 2s.; 1 chisel, value 1s.; 2 brushes, value 4s.; 3 straps, value 7s.; 1lb. weight of glue, value 7d.; 1 1/2lb. weight of chrome, value 1s.; 91bs. weight of nails, value 5s. 3 lamps, value 7s. 31 hooks, value 1s. 11 sheets of sand-paper, value 1s. 31bs. weight of iron, value 1s. 4 yards of serge, value 6s.; 1lb. weight of twine, value 1s.; 2 brooms, value 4s.; 1 set of fire-irons, value 5s. 2 brass rods, value 1s.; 1 lamp-burner, value 1s.; and 1 reflector, value d.—also, on the 
1st of January, 100 nails, value 3d.; the goods of the 

London and Birmingham Railway Company, his masters; to which he pleaded





GUILTY.Aged 37.—


Transported for Seven Years
