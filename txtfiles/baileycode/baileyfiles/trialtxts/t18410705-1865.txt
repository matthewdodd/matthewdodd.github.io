





1865. 




JOHN BALLINGER
 was indicted

 for burglariously breaking and entering the dwelling-house of 



Charles Leach, about the hour of two in the night, on the 
12th of May, at 

St. James's, Westminster, with intent to steal, and stealing therein 7 soldering irons, value 14s.; 1 hammer, value 1s.; 2 pairs of compasses, value 2s.; 1 pair of shears, value 3s.; 1 pair of pincers, value 1s. 6d.; 1 candlestick, value 5s.; 1 bottle-jack, value 10s.; 2 planishers, value 16s.; 1 stock and dies, value 15s.; 6 punches, value 6s.; 4 chisels, value 4s.; 1 screw-driver, value 2s.; 5lbs. weight of veal, value 3s.; 3lbs. weight of bacon, value 2s.; 2 brushes, value 2s.; 2 towels, value 2s.; and I screw-plate and taps, value 10s.; the goods of the said Charles Leach: 1 jacket, value 3s.; 1 pair of trowsers, value 3s.; 1 waistcoat, value 1s.; and 1 apron, value 2s.; the goods of 



Thomas Pritchard:—

Also for burglariously breaking and entering the dwelling-house of 



Charles Leach, about the hour of two in the night, on the 
5th of June, at 

St. James's, Westminster, with intent to steal, and stealing therein 2 pairs of shears, value 1l. 5s.; 42 punches, value 1l. 5s.; 3 hammers, value 30s.; 1 anvil, value 2s.; 1 pair of pliers, value 2s.; 3 soldering irons, value 5s.; 1 brush, value 1s.; 1lb. weight of solder, value 6d.; 1 blow-pipe, value 1s.; 1 basket, value 1s.; 2 towels, value 2s.; 1 chisel, value 1s.; and 1 screw-driver, value 1s.; the goods of the said Charles Leach:—

Also for burglariously breaking and entering the dwelling-house of 



Andrew Phillips, about the hour of three in the night, on the 
1st of May, at 

St. James's, Westminster, with intent to steal, and stealing therein 2 shirts, value 5s.; 1 coat, value 1l.; 1 umbrella, value 3s.; 2 pairs of boots, value 1l.; 1 handkerchief, value 2s.; 1 vice, value 4s.; 1 pair of pliers, value 2s.; 1 hammer, value 1s.; 2 files, value 2s.; 3 punches, value 2s.; 3 brushes, value 2s.; 1 pair of scissors, value 1s.; 1 pepper-box, value 6d.; 1 basket, value 1s. 6d.; 1 key, value 1d.; 5lbs. weight of lamb, value 3s.; 1lb. weight of butter, value 1s.; and 2lbs. weight of sugar, value 2s.; the goods of the said Andrew Phillips; to all of which he pleaded




GUILTY
. Aged 27.—


Transported for Life.
