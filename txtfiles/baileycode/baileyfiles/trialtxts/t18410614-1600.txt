




1600. 




ELIZABETH M'COLOUGH
 was indicted

 for stealing, on the 
2nd of June, at 
St. Luke, Chelsea, 13 yards of velvet, value 4l. 15s.; 27 rows of beads, value 7s.; 3 head-ornaments, value 12s.; 1 collar, value 8s.; 1 feather, value 6s.; 1 pair of socks, value 9d.; 4 handkerchiefs, value 4s.; 3 girdles, value 7s.; 28 tassels, value 12s.; 1 hair-brush, value 3s.; 24 yards of silk cord, value 3s.; 3 pairs of stockings, value 3s.; 1 pair of bracelets, value 5s.; 1 watch, value 2l. 10s.; 1 watch-guard, value 4s.; 1 seal, value 4s.; 1 watch-key, value 3s.; 3 rings, value 1l.; 1 bonnet, value 14s.; and 33 sovereigns; the property of 



Henry Meeking, her master, in his dwelling-house:—Also, 

for stealing, on the 
14th of May. 1 spoon, value St., the goods of 



Osborn Fothergill; to both of which she pleaded






GUILTY
. Aged 17—


Transported for Ten Years.
