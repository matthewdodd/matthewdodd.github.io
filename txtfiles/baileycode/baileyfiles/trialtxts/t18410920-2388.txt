



2388. 




WILLIAM SMITH
 was indicted

 for burglariously breaking and entering the shop of 



James Catling and another, on the 
25th of August, at 
St. John, at Hackney, and stealing therein 5 planes, value 15s.; 1 plough, value 7s.; 3 plough-irons, value 3s.; I stock,—value 3s.; 13 centre-bits, value 7s.; 3 plane-irons, value 3s.; 3 saws, value 10s.; 3 chisels, value 3s.; 1 gouge, value 5s.; 3 planes, value 5s.; 1 spoke-shave, value 1s.; 1 screw-driver, 1s.; 1 saw-set, value 1s.; the goods of 



Isaac Hawkins: and 1 plane, value 3s.; 1 saw, value 2s.; 4 chisels, value 3s.; 3 gouges, value 3s.; 2 gimlets, value 6d.; 4 brad-awls, value 4d.; and 2 squares, values 5s.; the goods of 



William Simmonds:—also for stealing, on the 
26th of August, 9 saws, value 2l. 9s.; 10 planes, value 2l. 15s.; 16 chisels, value 9s.; 1 bevil, value 3s.; 2 gouges, value 9s.; 3 hammers, value 3s.; 4 guagges, value 1s.; 2 screw-drivers, value 5s.; 2 spoke-shaves, value 3s.; 2 rasps, value 4s.; 1 file, value 1s.; 2 squares, value 5s.; 5 brad-awls, value 1s.; 6 gimlets, value 3s.; 2 rules, value 6s.; 2 pairs of pincers, value 2s.; 14 bits, value 7s.; and 3 baskets, value 3s.; the goods of 



Daniel Parker: to both of which he pleaded





GUILTY
. Aged 39.—


Transported for Fifteen Years.
